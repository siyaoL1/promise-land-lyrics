# Promise Land Lyrics

A lyrics website for the album *Promise Land*, featuring bilingual (Simplified/Traditional Chinese) lyrics with a dark, elegant design.

## Design Language

- Dark background with gold accent tones
- Corner art with gold-tinted decorative vectors
- Shimmer text effects and smooth fade transitions
- Responsive layout optimized for mobile
- Custom scrollbar and backdrop blur effects

## Structure

```
index.html          — Single-page app entry point
css/styles.css      — All styles (layout, animations, responsive)
js/app.js           — Landing page, navigation, search
js/lyrics.js        — Lyrics rendering, font size, character toggle
data/songs.json     — Song data with bilingual lyrics
assets/             — Album poster, corner art, fonts
```

## Features

- Search songs by title or lyrics content
- Simplified/Traditional Chinese toggle (简/繁)
- Adjustable font size (A−/A+)
- Smooth page transitions with fade effects
- Mobile-friendly with touch-optimized interactions
